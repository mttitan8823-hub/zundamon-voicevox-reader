# 🔊 ずんだもん読み上げ拡張機能 (VOICVOX Reader)

この拡張機能は、ブラウザで選択したテキストを、ローカルで実行している [VOICVOX Engine](https://voicevox.hiroshiba.jp/) の **ずんだもん（ノーマル）音声** で読み上げるための Google Chrome / Brave 向けの拡張機能です。

音声処理をローカルで完結させるため、外部サービスにテキストデータを送信しません。

---

## ✨ 機能と利点

* **右クリックメニュー:** 選択したテキストを直接、ずんだもん音声で読み上げます。
* **リアルタイム設定:** ツールバーのアイコン（ポップアップ）から、**音量 (Volume)** と **話す速度 (Speed)** を即座に調整し、その設定を保存します。
* **プライバシー:** 外部ネットワークを経由せず、PC内のVOICVOXエンジンで音声を合成します。

---

## 🛠️ 動作環境と前提条件

この拡張機能を利用するためには、以下の環境が必須となります。

1.  **VOICVOX Engine の起動**
    * PC上でVOICVOX Engine 本体が起動していること。
    * Engineが**ローカルポート `50021`** で待ち受けていること。
    * 使用スピーカーID: **`3`** (ずんだもん / ノーマル)

2.  **ブラウザ**
    * Google Chrome または Brave Browser

---

## 📥 インストール手順

以下の手順に従い、拡張機能をブラウザに読み込ませてください。

1.  **リポジトリのダウンロード**
    * GitHubのページで緑色の「**Code**」ボタンをクリックし、「**Download ZIP**」を選択して、リポジトリ全体をダウンロードします。
    * ダウンロードしたZIPファイルを展開してください。

2.  **ブラウザの準備**
    * ブラウザのアドレスバーに `chrome://extensions` と入力して拡張機能管理ページを開きます。
    * 画面右上の「**デベロッパーモード (Developer mode)**」をオンにします。

3.  **拡張機能の読み込み**
    * 「**パッケージ化されていない拡張機能を読み込む (Load unpacked)**」ボタンをクリックします。
    * 手順1で展開したフォルダ（例: `zundamon-voicevox-reader-main`）を選択し、読み込みます。
